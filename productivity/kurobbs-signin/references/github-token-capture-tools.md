## ProxyPin下载
- 最新Release: `https://github.com/wanghongenpin/proxypin/releases/latest`
- APK文件名: `proxypin-android.apk`
- GitHub镜像: `gh-proxy.com` 可用（mirror.ghproxy.com已失效）
- 下载命令: `curl -L -o ProxyPin.apk "https://gh-proxy.com/<release_url>"`
